# PunchClock

Free browser-based time clock for freelancers. Clock in, clock out, export your hours to CSV. No signup, no account, no bloat.

**Live:** https://punchclock.visieasy.com

## Features

- One-click punch in/out with live timer
- Daily and weekly hour totals
- Export all entries to CSV
